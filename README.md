# 大一程序设计项目合集
C 语言课程实践项目 | 控制台应用 | 数据结构与程序逻辑训练

---

## 项目列表

### 🏦 银行管理系统
- 路径：`/bank`
- 语言：C 语言
- 功能：
  - 用户注册、登录、密码修改
  - 存款、取款、余额查询
  - 交易记录保存与查看
  - 文件持久化存储数据
- 简介：模拟简易银行柜台业务，综合运用结构体、文件操作、菜单交互逻辑。

---

### ✈️ 空战小游戏
- 路径：`/kongzhan`
- 语言：C 语言
- 功能：
  - 控制飞机移动、发射子弹
  - 敌机自动生成与移动
  - 计分系统、生命值判断
  - 控制台画面刷新
- 简介：控制台图形小游戏，练习循环、坐标计算、按键检测与简单碰撞逻辑。

---

### 🔍 字段查询系统
- 路径：`/lisanshuxue`
- 语言：C 语言
- 功能：
  - 读取结构化文本数据
  - 按字段精准/模糊查询
  - 信息筛选、格式化输出
  - 简单数据管理
- 简介：结构化数据查询工具，训练字符串处理、数组与查找算法。

---

## 使用说明
所有项目均为控制台程序，使用 C 语言编译器（如 GCC、Dev-C++、VS）编译运行即可。
